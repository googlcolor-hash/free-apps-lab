2) PowerToys: насколько нам нужно

Итог: НУЖНО (High priority) для вашей рабочей модели с большим количеством локальных операций, окон и файловых потоков.

Опирался на официальные страницы Microsoft Learn и GitHub PowerToys (актуальные модули: FancyZones, PowerRename, Keyboard Manager, Awake, Command Palette, Peek и др.).

МодульНужностьЗачем в вашем процессе
FancyZonesКритичноБыстрые layout-и под параллельные окна Codex/браузер/терминал.
PowerRenameКритичноМассовые переименования ассетов, экспортов, скриншотов и архивов.
Keyboard ManagerВысокоУскорение частых хоткеев и выравнивание UX между приложениями.

Peek + File LocksmithВысокоБыстрый предпросмотр и диагностика файловых блокировок.
